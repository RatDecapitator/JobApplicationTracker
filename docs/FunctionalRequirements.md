# JobApplicationTracker Functional Requirements 

FR-1.1: User can add a new job application by providing: company name, job title, location, date of application, and the CV used.
    FR-1.2: App tracks and displays the elapsed time since each application was submitted.

FR-2.1: User can log a response from a company, including the exact date of the response and its content.
    FR-2.2: User can update the application status based on the response (e.g., Pending, In Progress/Interview, Accepted/Offer, Failed/Rejected).

FR-3.1: App displays the percentage of failed, accepted, and pending applications.
    FR-3.2: App displays the average response time from companies.

FR-4.1: App tracks how many times each specific CV was used.
    FR-4.2: App displays the success rate (percentage of failed/accepted/pending) for each CV used.
